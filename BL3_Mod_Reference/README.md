# BL3_Mod_Reference

Thư viện tham chiếu cho Borderlands 3 modding, dựa trên data observed từ bd3miner và bl3data.

## Cấu trúc
- sdk/classes.json: Class paths và metadata từ observed events.
- events/inventory_events.json: Lịch sử events (item pickup, etc.).
- (Tương lai: data/ cho bl3data, flow/ cho docs, notes/ cho quirks).

## Cách sử dụng
Load JSON files trong mod để reference class paths hoặc events.
